# Kiraly expense-tracker-api

## Description

Api for Kiraly expense-tracker app

###

[@yoiler_cordoba](https://x.com/yoiler_cordoba)

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
